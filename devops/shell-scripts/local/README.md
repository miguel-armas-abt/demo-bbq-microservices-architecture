## 📋 Pre requisitos
> ⚙️ **Instalar herramientas**<br>
> `Java 17+`, `Maven 3.9.1+`, `GO 1.21+`, `Kafka`, `MySQL`, `PostgreSQL`, `Prometheus`, `Zipkin`, `Grafana`, `Loki`, `yq`
>
> ⚙️ **Configurar rutas de instalación**
> ```shell script 
> notepad ./../../../application/local.csv
> ```

> - Compilar proyectos

> - Ejecutar servicios

> ./main.sh
> ```
